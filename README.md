React-test
=

## 介绍

这是一个react测试项目,旨在通过webpack打包生成用于restful API的one page react项目

## 技术栈
```
*babel
*webpack
*react
*redux
*react-redux
*react-router
*react-router-redux
```

## 目录结构
```
-client       //客户端项目库
--app.js      //入口文件
--router.js   //路由
---actions    //redux action列表
---reducers   //redux reducer列表
---components //react组件
---pages      //页面组件
-public       //发布目录
```
